pfam data for training the language model and data split used can be downloaded directly from http://s3.amazonaws.com/songlabdata/proteindata/data_pytorch/pfam.tar.gz or from the TAPE-Protein repository (https://github.com/songlab-cal/tape).

The downloaded files include: pfam_holdout.lmdb,  pfam_strings.txt,  pfam_train.lmdb,  pfam_valid.lmdb
